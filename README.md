📝 파이썬 프로그래밍 과제 #32~#38
이 저장소는 파이썬의 set 자료형을 활용한 과제 모음입니다. 각 문제는 집합 연산, 메서드 활용, 조건 처리, 반복문 등을 통해 set을 다루는 연습을 목표로 합니다.

✅ 과제 32
- 본인이 좋아하는 숫자 10개를 5개씩 두 개로 나누어 set을 생성
- 두 집합의 **합집합(union)**과 **교집합(intersection)**을 계산하여 출력

✅ 과제 33
- 본인이 좋아하는 숫자 10개를 5개씩 두 개로 나누어 set을 생성
- 두 집합의 **차집합(difference)**과 **대칭차집합(symmetric difference)**을 계산하여 출력

✅ 과제 34
- 본인이 좋아하는 숫자 5개로 set을 생성
- update 메서드를 사용하여 숫자 10을 포함시킨 후 출력

✅ 과제 35
- a = {100, 200, 300, 400, 500}과 b = {400, 500, 600, 700, 800} 두 집합을 선언
- intersection_update, difference_update, symmetric_difference_update 메서드를 각각 수행하고 결과 출력

✅ 과제 36
- a = {100, 200, 300, 400, 500}과 b = {100, 200, 300, 400, 500, 600} 두 집합을 비교
- a가 상위집합이면 "상위" 출력
- a가 부분집합이면 "부분" 출력
- 둘 다 해당되면 "동시" 출력

✅ 과제 37
- 본인이 좋아하는 숫자 5개로 set을 생성
- 숫자 1000을 add 메서드로 추가한 후
- pop 메서드로 마지막 값을 삭제하고 출력

✅ 과제 38
- 1부터 100까지 숫자 중
- 3과 5의 공배수를 set comprehension을 이용해 추출
- 결과를 set 형태로 출력
