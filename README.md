# 파이썬 프로그래밍 과제 #25~#31

이 저장소는 파이썬 딕셔너리(dictionary)를 활용한 과제 모음입니다. 각 문제는 입력, 출력, 조건 처리, 반복문 등을 통해 딕셔너리를 다루는 연습을 목표로 합니다.

## 📝 과제 목록

### ✅ 과제 25
- 키와 값을 입력받아 딕셔너리 생성
- 특정 키들만 추출하여 출력

### ✅ 과제 26
- `park` 딕셔너리 선언
- `'english'`와 `'science'` 점수 출력

### ✅ 과제 27
- `kim` 딕셔너리 선언
- 모든 과목 점수를 100점으로 수정 후 출력

### ✅ 과제 28
- `lee` 딕셔너리 선언
- `'english'` 키가 있으면 삭제 후 출력

### ✅ 과제 29
- `lim` 딕셔너리 선언
- 모든 키와 값 출력

### ✅ 과제 30
- `choi` 딕셔너리 선언
- 90점 이상인 과목의 키만 출력 (dictionary comprehension 사용)

### ✅ 과제 31
- `yoo` 딕셔너리 선언
- 4과목 평균 점수 출력
